Target Users
------------

XPCS Toolkit is designed for:

- X-ray scientists analyzing photon correlation spectroscopy data
- Beamline users at synchrotron facilities (particularly APS 8-IDI)
- Researchers working with time-resolved scattering measurements
- Data analysts processing large XPCS datasets

Prerequisites
-------------

- Basic Python programming knowledge
- Understanding of X-ray scattering principles
- Familiarity with HDF5 data format
- Knowledge of correlation function analysis concepts
